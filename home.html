    About project

    In order to build the application several Java elements like JLabels, JButtons, JTextFields, JMenuItems or Arrays were used.
    To create the interface in which the car navigates, a loop was used to create two hundred and eight buttons that represent a grid, a grid that becomes a maze by using a one-dimensional array and five methods that represent the worlds, in each world the car will get a different icon as well as the compass. The purpose of the game is to get the car from one place to another by pressing a button, making it navigate through the maze at a speed defined by a slider. Numerous if and else statements were used to apply conditions and to check every step of the car that moves one block at a time. The car can’t go through walls and dialog message boxes are displayed on the screen if the user tries to do so.
    A second option was developed using a different dimensional array, in this way five different worlds were implemented, the code checks if the "Option 2" button was pressed and change the maze if the Boolean variable is false. The second world presents obstacles as kids and the car has a different routine for completing a lap.
